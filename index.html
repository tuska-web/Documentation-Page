Introduction

C++ is a powerful general-purpose programming language that supports procedural, object-oriented, and generic programming. It is widely used for system/software development, game development, and real-time systems.

This documentation provides an overview of C++ features and syntax.

Syntax

C++ syntax is similar to C but includes object-oriented features. The basic structure of a C++ program is as follows:


#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
Data Types

C++ supports various data types, including:

  • int: Represents integers.
  • float: Represents floating-point numbers.
  • double: Represents double-precision floating-point numbers.
  • char: Represents single characters.
  • bool: Represents boolean values (true or false).
Control Structures

C++ supports various control structures such as if-else, switch-case, for loops, while loops, and do-while loops. Example of a for loop:


for (int i = 0; i < 10; i++) {
    cout << i << " ";
}
Functions

Functions allow you to encapsulate code and reuse it. Example of a function:


int add(int a, int b) {
    return a + b;
}

int main() {
    int sum = add(5, 3);
    cout << "Sum: " << sum << endl;
    return 0;
}
Classes

Classes are the foundation of object-oriented programming in C++. They allow you to define your own data types and include both data (attributes) and functions (methods) that operate on the data. Example of a class:


class Rectangle {
public:
    int width, height;

    Rectangle(int w, int h) {
        width = w;
        height = h;
    }

    int area() {
        return width * height;
    }
};

int main() {
    Rectangle rect(10, 5);
    cout << "Area: " << rect.area() << endl;
    return 0;
}
Pointers

Pointers are a powerful feature in C++ that allow you to directly access memory and manipulate addresses. Example of using pointers:


int main() {
    int var = 5;
    int *ptr = &var;  // Pointer to var

    cout << "Value of var: " << var << endl;
    cout << "Address of var: " << ptr << endl;
    cout << "Value at address ptr: " << *ptr << endl;

    return 0;
}
Standard Template Library (STL)

The Standard Template Library (STL) is a powerful library in C++ that provides a wide range of generic classes and functions, such as vectors, lists, and algorithms. Example of using a vector from the STL:


#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> vec = {1, 2, 3, 4, 5};

    // Iterate and print values
    for(int i : vec) {
        cout << i << " ";
    }

    return 0;
}
Conclusion

C++ is a versatile and powerful programming language that provides a range of features for system programming, application development, and more. Its support for object-oriented programming, generic programming through templates, and robust standard library makes it a go-to language for many complex and performance-critical applications. With a solid understanding of C++'s syntax, data types, control structures, functions, classes, pointers, and the Standard Template Library (STL), you can develop efficient and effective software solutions.
